Penjelasan Materi: Pengantar HTML

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. Dalam materi ini, kita akan belajar mengenai elemen-elemen dasar HTML seperti paragraf, heading, dan link.

Setelah mempelajari materi ini, kamu akan dapat memahami struktur dasar halaman web dan mulai menulis kode HTML untuk membuat website.
